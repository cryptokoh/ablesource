# Ablesource Website Styles

A collection of 4 distinctive website designs for Ablesource HR & Compliance services.

## Live Demo

Visit the [Style Gallery](https://ablesource.netlify.app) to preview all designs.

## Available Styles

| Style | Description |
|-------|-------------|
| **Corporate Editorial Luxury** | Sophisticated serif typography, muted earth tones, refined elegance |
| **Bold Brutalist** | High-contrast industrial aesthetic with electric accents |
| **Warm Organic** | Friendly, approachable feel with soft gradients |
| **Geometric Art Deco** | Sharp precision with gold accents on deep navy |

## Local Development

Simply open `index.html` in a browser, or use a local server:

```bash
npx serve .
```

## Deployment

This site is configured for Netlify. Simply connect the GitHub repo to Netlify for automatic deployments.

## Files

- `index.html` - Style gallery/selector
- `style1-editorial-luxury.html` - Corporate Editorial style
- `style2-brutalist.html` - Bold Brutalist style
- `style3-warm-organic.html` - Warm Organic style
- `style4-art-deco.html` - Geometric Art Deco style
- `logo.png` - Ablesource logo (white)
- `netlify.toml` - Netlify configuration
